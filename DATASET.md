# Dataset Information

This project uses publicly available datasets for detecting AI-generated images.

## CIFAKE Dataset

- Description: Binary classification dataset containing real and AI-generated images.
- Source: Kaggle  
- Link: https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images  


## GenImage Dataset

- Description: Large-scale dataset containing images generated from multiple generative models.
- Source: GitHub  
- Link: https://github.com/GenImage-Dataset/GenImage
